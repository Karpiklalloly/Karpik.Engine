# 0.2
## UI
Положил начало для интерфейса (вайб код помог). На данный момент реализовано:

**Форматы значений:**
*   **`<length>`**: Числовое значение. Может быть указано как строка с `px` (`"10px"`) или просто число (`10`).
*   **`<percent>`**: Значение в процентах, указанное как строка (`"50%"`).
*   **`<color>`**: Имя цвета в виде строки (например, `"white"`, `"red"`, `"lightblue"`). Поддерживается `transparent`.
*   **`<number>`**: Число без единиц измерения (используется во `flex`).
*   **`<integer>`**: Целое число (используется в `z-index`).

### Компоновка и Позиционирование (Layout & Positioning)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`display`** | `flex` \| `block` \| `none` | Определяет, как элемент участвует в компоновке. `flex` активирует Flexbox. |
| **`position`** | `static` \| `relative` \| `absolute` \| `fixed` | Устанавливает тип позиционирования элемента. |
| **`top`** | `<length>` \| `<percent>` \| `auto` | Задает смещение для позиционированных элементов от верхнего края. |
| **`right`** | `<length>` \| `<percent>` \| `auto` | Задает смещение для позиционированных элементов от правого края. |
| **`bottom`**| `<length>` \| `<percent>` \| `auto` | Задает смещение для позиционированных элементов от нижнего края. |
| **`left`** | `<length>` \| `<percent>` \| `auto` | Задает смещение для позиционированных элементов от левого края. |
| **`z-index`** | `<integer>` \| `auto` | Управляет порядком наложения (стекирования) позиционированных элементов. |
| **`overflow`**| `hidden` | Определяет поведение контента, выходящего за границы элемента. *Поддерживается только `hidden`.* |

### Размеры (Sizing)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`width`** | `<length>` \| `<percent>` \| `auto` | Задает ширину элемента. |
| **`height`**| `<length>` \| `<percent>` \| `auto` | Задает высоту элемента. |

### Flexbox-компоновка (для `display: flex` контейнеров)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`flex-direction`**| `row` \| `column` | Определяет направление главной оси (горизонтальное или вертикальное). |
| **`flex-wrap`**| `nowrap` \| `wrap` | Разрешает или запрещает перенос flex-элементов на новую строку. |
| **`justify-content`**| `flex-start` \| `flex-end` \| `center` \| `space-between` \| `space-around` | Выравнивает элементы вдоль главной оси. |
| **`align-items`** | `flex-start` \| `flex-end` \| `center` \| `stretch` | Выравнивает элементы вдоль поперечной оси (внутри одной строки). |
| **`align-content`** | `flex-start` \| `flex-end` \| `center` \| `space-between` \| `space-around` \| `stretch` | Выравнивает строки в многострочном flex-контейнере. |
| **`flex-grow`** | `<number>` | Определяет, какую долю свободного пространства может занять элемент. |
| **`flex-shrink`**| `<number>` | Определяет, насколько элемент может сжиматься при нехватке места. |
| **`flex-basis`** | `<length>` \| `<percent>` \| `auto` | Задает базовый размер элемента перед распределением свободного пространства. |
| **`align-self`** | `auto` \| `flex-start` \| `flex-end` \| `center` \| `stretch` | Позволяет переопределить `align-items` для отдельного flex-элемента. |

### Box Model (Отступы и границы)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`margin`** | `<length>` \| `<percent>` | Задает внешние отступы. Поддерживаются `margin-top`, `margin-right` и т.д. |
| **`padding`** | `<length>` \| `<percent>` | Задает внутренние отступы. Поддерживаются `padding-top`, `padding-right` и т.д. |
| **`border-width`** | `<length>` | Задает толщину границы. Поддерживаются `border-top-width`, `border-right-width` и т.д. |
| **`border-color`** | `<color>` | Задает цвет границы. *На данный момент стиль (solid, dashed) не поддерживается.* |
| **`box-sizing`** | `content-box` \| `border-box` | Определяет, как `width` и `height` рассчитываются: включая `padding` и `border` или нет. |

### Внешний вид и Отрисовка (Appearance & Rendering)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`background-color`** | `<color>` \| `transparent` | Задает цвет фона элемента. |
| **`color`** | `<color>` | Задает цвет текста. |

### Текст и Шрифты (Text & Fonts)

| Свойство | Значения | Описание |
| :--- | :--- | :--- |
| **`font-size`** | `<length>` | Задает размер шрифта. |
| **`line-height`** | `<length>` \| `<percent>` \| `auto` | Задает высоту строки для текста. |
| **`text-align`** | `left` \| `center` \| `right` | Задает горизонтальное выравнивание текста. |
