# Input

> Модуль обработки пользовательского ввода

## 📋 Обзор

- **Слой**: Client
- **Приоритет**: 0 (стандартный)
- **Интерфейсы**: `IModule`, `IModuleConfiguratable`

## 🎯 Назначение

Обрабатывает ввод с клавиатуры и мыши, предоставляет унифицированный API для игровых систем.

## 📦 Сервисы

| Тип | Описание |
|-----|----------|
| `Input` | Главный класс управления вводом |

## 🔧 ECS-системы

| Система | Слой | Описание |
|---------|------|----------|
| `UpdateSystem` | Update | Обновление состояния ввода |

## 📁 Структура

```
Input/
├── InputInstaller.cs      # Инсталлер модуля
├── Input.cs               # Главный класс ввода
├── InputModuleEcs.cs      # ECS-модуль
├── KeyboardKeys.cs        # Коды клавиш
├── MouseButtons.cs        # Коды кнопок мыши
└── Systems/
    └── UpdateSystem.cs    # Система обновления
```

## 🔗 Зависимости

- `Graphics.Core` — для получения `IWindow`
- Raylib — для нативного ввода

## 💡 Использование

```csharp
// Получение сервиса
var input = services.Get<Input>();

// Проверка нажатия
if (input.IsKeyDown(KeyboardKeys.W))
{
    // Движение вперёд
}
```

## ⚠️ Особенности

- Требует инициализацию через `IWindow`
- Автоматическое обновление каждый кадр
- Поддержка клавиатуры и мыши
